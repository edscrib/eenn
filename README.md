E-Commerce Sales & Customer Analytics Dashboard
Overview

This project analyzes transactional sales data from an e-commerce retailer to uncover insights into customer behavior, product performance, and revenue trends. The objective is to demonstrate end-to-end Business Intelligence and Data Analytics skills, including data cleaning, SQL analysis, KPI development, and interactive dashboard creation.

Business Problem

Retail businesses generate large amounts of transactional data but often struggle to transform it into actionable insights. This project addresses key business questions:

Which products generate the highest revenue?
Who are the most valuable customers?
How does revenue evolve over time?
Which countries contribute the most to sales?
How can customers be segmented for targeted marketing?
Project Objectives
Clean and prepare raw transactional data.
Store and analyze data using PostgreSQL.
Develop SQL queries to answer business questions.
Create KPIs for executive decision-making.
Build an interactive BI dashboard.
Perform customer segmentation using RFM Analysis.
